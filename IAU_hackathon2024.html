Welcome!

This workshop is part of the focus meeting "New horizons at the interface between computational astrophysics and big data". In this hackathon: "A first taste of MESA - an open source tool box for experiments in stellar astrophysics", we will explore some of the essentials of the open-source stellar evolution code MESA .

★ Created by: Lieke van Son, Shelley Cheng, and Aleksandra Olejak ★

Before getting started

In this tutorial, we have pre-installed MESA for you on the _flatiron_ cluster. If you would like to install MESA on your own computer or cluster, you can find the installation instructions here.

You will be able to access the cluster through the Binder environment. For this, you need to log in using a Google account. This can be any Google account, including a personal one.
To gain access to the Binder environment, please send the Gmail address you will use to log in to lvanson_at_flatironinstitute.org so we can add you to the list of approved users!

How to use the Binder environment

Once we've added you to the list of approved users, you can access the Binder environment through the following link: https://sdsc-binder.flatironinstitute.org/~lvanson/MESAhackathon
General instructions on how to use this Binder environment can be found on the Flatiron institute wiki.

Setting up your environment for MESA

Open a terminal and type the following command to set up the environment for MESA usage: 
        
# set OMP_NUM_THREADS to be the number of cores available on your machine
export OMP_NUM_THREADS=16

# Mesa root
export MESA_DIR='/mnt/home/lvanson/Programs/MESA/mesa-r24.03.1'

# Mesa sdk
export MESASDK_ROOT='/mnt/home/lvanson/Programs/MESA/mesasdk_23051/mesasdk'
source $MESASDK_ROOT/bin/mesasdk_init.sh
Note, normally, you would add these lines to your .bashrc file so that they are automatically executed every time you log in. See also the MESA installation instructions here.
